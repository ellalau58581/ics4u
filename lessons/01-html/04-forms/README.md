# 🟩 HTML Forms

## 🚀 Features

- Understanding how HTML forms collect user input
- Using different input types for various data formats
- Working with selection inputs like radio buttons, checkboxes, and dropdowns
- Using buttons for form actions
- Using special input types like file upload, color picker, and range slider
- Structuring form content using `<form>` and grouping inputs

## 🧠 Key Concepts

### Text Inputs

Text-based inputs allow users to enter free-form data:

- `text` → general text input
- `email` → email address input
- `password` → hidden text input
- `number` → numeric input
- `tel` → phone number input
- `url` → website link input
- `search` → search field

### Date & Time Inputs

Used for selecting dates and times:

- `date` → calendar date picker
- `time` → time selector
- `datetime-local` → date and time combined
- `month` → month and year
- `week` → week selector

### Selection Inputs

Used for choosing between options:

- `radio` → single choice selection (same `name` groups them)
- `checkbox` → multiple selections allowed
- `select` + `option` → dropdown menu

### Buttons

Buttons trigger actions:

- `button` → generic button
- `submit` → submits the form
- `reset` → resets form values

### Special Inputs

Advanced input types:

- `file` → upload files
- `color` → color picker
- `range` → slider input
- `hidden` → hidden data field (not visible to user)

### Textarea

- `<textarea>` → multi-line text input

### Form Structure

- `<form>` wraps all input elements
- Inputs inside a form are grouped together for submission
