# 🚀 Assignment 4 Demo

## 1. Open in VS Code

- Open the **assign4** folder in Visual Studio Code

## 2. Install Live Server

- Open Extensions (`Ctrl + Shift + X`)
- Search for **Live Server** by _Ritwick Dey_
- Install it

## 3. Run the project

- Click **Go Live** at the bottom-right of VS Code, but

> [!WARNING]
> Make sure no files are open before doing the above step.

## 🌐 What happens

- The site opens in your browser
- You will be prompted to enter your **TMDB API key**
- Click **“Set Key”** to save it
- Click **“Enter”** to access the site

> [!TIP]
> You can change your API key by repeating the above step.
